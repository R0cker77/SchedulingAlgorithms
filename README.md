# SchedulingAlgorithms
Some simple Scheduling Algorithms made in python


Algorithms:
First come, first served
Shortest Job First
Priority Queue
Round Robin

side notes:
All alogrithms are None Preemptive
In round robin the quantim time as defualt is 5.
